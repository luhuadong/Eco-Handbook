可持续软件工程
===========================================

可持续软件工程（Sustainable Software Engineering）是一门结合了气候科学、软件、硬件、电力市场和数据中心设计的新兴学科。 可持续软件工程的原则和理念是定义、构建和运行可持续软件应用程序所需的一系列核心能力。 通过综合这些知识，可持续软件工程师 (SSE) 可以做出对应用程序碳污染有重大影响的决策。
